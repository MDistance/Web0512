# 测试

实现以下需求

GET  /login   返回一个表单
```html
<form action="/login" method="post">
    <input type="text" name="username">
    <input type="text" name="password">
    <button>点击提交</button>
</form>
```

POST /login  将表单数据以表格形式呈现

| username | 用户名的值 |
| -------- | ---------- |
| password | 密码的值   |
